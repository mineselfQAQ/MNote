- 十进制 Decimal
- 二进制 Binary
- 十六进制 Hexadecimal

# 十转二

<B>方法：</B>

``` csharp
List<int> bits = new List<int>();

while(n > 0)
{
    bits.Add(n % 2);
    n /= 2;
}
```

其实只有一个<B>重点</B>：
<B><VT>/2 后余数只可能为0或1，且是最低位</VT></B>

<B>数学推导：</B>
任意一个数的二进制可拆解为：
$$
a_0 + a_1 \cdot 2 + a_2 \cdot 2^2 + a_3 \cdot 2^3
$$
所以：
$$
n = a_0 + 2(a_1 + a_2 \cdot 2 + a_3 \cdot 2^2 + \dots)
$$
由此可得：
$$
n \bmod 2 = a_0 \\
n / 2 = a_1 + a_2 2 + a_3 2^2
$$

# 二转十

<B>方法：</B>

``` csharp
int sum = 0;

for(int i = 0; i < bits.Count; i++)
{
    sum += bits[i] * (1 << i);
}
```

二转十会比十转二好理解很多：
从最低位开始，根据值（0/1）进行二的幂运算，因为：
$$
... a_3 a_2 a_1 a_0
$$
实际上：
$$
... + a_3 \cdot 2^3 + a_2 \cdot 2^2 + a_1 \cdot 2^1 + a_0 \cdot 2^0
$$
