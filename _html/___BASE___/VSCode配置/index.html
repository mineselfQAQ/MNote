VSCode配置

C#

在VSCode中，如果要配置C#的话是略有麻烦的，具体如下：
在VSCode的配置前，首先需要去官网下载.NET即可，一般选择长期支持版(目前是8.0)
在VSCode中，扩展肯定是需要添加的，有：

  • C# (Microsoft)
  • C# Dev Kit (Microsoft)
    可以Shift+Ctrl+P输入walkthrough选择Welcome:Open Walkthrough并选择C# Dev Kit即可查看全流程

具体构建流程：

  1. 创建一个文件夹，在其中打开VSCode
  2. Shift+Ctrl+P输入.NET:Project，选择Console App并一路确认即可
  3. 然后就能看到创建的文件，打开Program.cs，通过F5运行即可
