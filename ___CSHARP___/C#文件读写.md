# 简述

文件操作是C#中的一个重要操作，相关内容就在System.IO中

核心内容大致可以分为以下部分：

- 环境
  - Environment
- 文件与目录
  - File/FileInfo
  - Directory/DirectoryInfo
  - Path
- 流
  - FileStream
  - IsolatedStorageFileStream
  - MemoryStream
  - BufferedeStream
  - NetworkStream
  - PipeStream
  - CryptoStream
- 读取器与编写器
  - BinaryReader/BinaryWriter
  - StreamReader/StreamWriter
  - StringReader/StringWriter
  - TextReader/TextWriter
