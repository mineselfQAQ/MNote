<center><B><BBBG>Git规则</BBBG></B></center>

- 日常提交标题
  `[xxxx_xx_xx][tag] 内容`
  - tag：
    - [feature]：新功能（feature）
    - [fix]：修复 bug
    - [refactor]：重构（不改功能，只优化代码）
    - [docs]：文档相关
    - [test]：测试相关（新增/修改测试）
    - [chore]：杂项（构建、依赖、工具配置等）
    - [style]：代码格式（不影响逻辑，比如空格、lint）
    - [perf]：性能优化
    - [build]：构建系统或打包相关
    - [ci]：CI/CD 配置
